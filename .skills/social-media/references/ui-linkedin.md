# LinkedIn Reference

Package: `com.linkedin.android`

## App Traits

- Professional networking and job search
- Post feed with articles and updates
- 6 reaction types (long-press to select)
- Job listings and applications
- Messaging for networking
- Bottom tab navigation with center (+) for posting

## Known Quirks

- **Reactions**: Long-press Like button for 6 options (Like, Celebrate, Support, Love, Insightful, Funny)
- **Connect vs Follow**: Connect sends invitation, Follow is one-way
- **InMail**: Premium feature for messaging non-connections
- **Creator mode**: Changes profile layout for content creators
- **Easy Apply**: One-click job applications
