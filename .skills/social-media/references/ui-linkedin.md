# LinkedIn UI Reference

Package: `com.linkedin.android`

## Navigation

Bottom tabs:
```
[Home] [My Network] [Post] [Notifications] [Jobs]
```

## Interaction Buttons

| Icon | Action |
|------|--------|
| Thumbs up | Like/React |
| Bubble | Comment |
| Loop arrows | Repost |
| Paper plane | Send to connection |

## Reactions (6 types)

Long-press Like to select:
1. Like (thumbs up)
2. Celebrate (clapping)
3. Support (heart hands)
4. Love (heart)
5. Insightful (lightbulb)
6. Funny (laughing)

## Key Actions

### React to Post
1. Tap Like for quick like
2. Long-press for reaction menu
3. Select reaction type

### Comment
1. Tap Comment button
2. Type in input field
3. Tap Post

### Repost
1. Tap Repost button
2. Select: Repost (instant) or Repost with thoughts

### Send Post
1. Tap Send button
2. Select connection(s)
3. Add message (optional)
4. Send

### Create Post
1. Tap [+] Post in bottom nav
2. Or tap "Start a post" at top
3. Write content
4. Add photo/video/document
5. Tap Post

### Connect
1. Go to profile
2. Tap Connect button
3. Optionally add note
4. Send

### Message
1. Tap Messaging icon (top)
2. Select conversation or compose new
3. Type and send

### Search
1. Tap search bar
2. Type query
3. Filter: People, Jobs, Posts, Companies, Groups

### My Network
1. Tap My Network tab
2. View suggestions
3. See pending invitations
4. Manage connections

### Jobs
1. Tap Jobs tab
2. Search jobs
3. Apply directly or save
