## Key Actions

### Send Message
1. Tap Chats tab
2. Select conversation
3. Tap input field
4. Type message
5. Tap send (green arrow)

### Send Voice Message
1. In chat, hold mic button
2. Swipe up to lock
3. Release or tap send when done

### Reply to Specific Message
1. Swipe right on message
2. Or long-press > Reply
3. Type reply

### React to Message
1. Long-press message
2. Select reaction from popup

### Send Media
1. Tap [+] or camera icon
2. Select: Document, Camera, Gallery, Audio, Location, Contact

### View Once Media
1. Tap [+] > Gallery
2. Select photo/video
3. Tap "1" icon for view once

### Voice/Video Call
1. Open chat
2. Tap call icon (voice) or video icon

### Status/Updates
1. Tap Updates tab
2. View contacts' status
3. Tap [+] to add your status

### Search
1. Tap search icon
2. Search messages, contacts, media

## Special Features

- End-to-end encryption
- Message editing (within time limit)
- Message deletion for everyone
- Broadcast lists
- Starred messages
- Chat wallpapers
