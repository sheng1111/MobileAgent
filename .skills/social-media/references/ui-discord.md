```

```

## Tabs Overview

### Servers Tab
- View all joined servers (vertical list)
- Tap server to see channels
- Tap channel to enter

### Messages Tab
- All DMs and group chats
- Friend activity carousel (what friends are doing)
- Search through messages

### Notifications Tab
- Mentions and replies
- Server notifications
- Filter by type

### You Tab
- Friends list
- Settings access
- Profile management

## Key Actions

### Send Message
1. Enter channel or DM
2. Tap input field
3. Type message
4. Tap send or press Enter

### Reply to Message
1. Swipe right on message
2. Or long-press > Reply
3. Type reply

### React to Message
1. Long-press message
2. Select reaction from popup
3. Or tap existing reaction to add

### Send DM
1. Go to Messages tab
2. Tap conversation or compose new
3. Type and send

### Join Voice Channel
1. Navigate to server
2. Find voice channel
3. Tap to join
4. Tap again to disconnect

### Create Server
1. Tap [+] in server list
2. Choose template or create custom
3. Set name and create

### Add Friend
1. Go to You tab > Friends
2. Tap Add Friend
3. Enter username#tag

### Search
1. Tap search icon
2. Search messages, servers, or users
