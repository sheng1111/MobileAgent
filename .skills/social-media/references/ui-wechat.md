## Key Actions

### Send Message
1. Tap WeChat tab
2. Select conversation
3. Tap input field
4. Type message
5. Tap Send button

### Send Voice Message
1. In chat, tap Voice button (left of input)
2. Hold to record
3. Release to send

### Moments (pengyouquan)
1. Tap Discover tab
2. Tap Moments (first item)
3. Like: double-tap or tap heart in comment area
4. Comment: tap in comment area

### Scan QR
1. Tap [+] icon (top-right)
2. Select Scan
3. Point camera at QR code

### Add Contact
1. Tap [+] icon
2. Select Add Contacts
3. Search by WeChat ID or phone

### Video/Voice Call
1. Open chat
2. Tap [+] in input area
3. Select Video Call or Voice Call

## Special Features

- Long-press message: recall (within 2 min), forward, copy
- Stickers: tap emoji icon
- Red packets: in [+] menu
- Transfer money: in [+] menu
