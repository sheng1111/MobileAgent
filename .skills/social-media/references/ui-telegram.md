# Telegram Reference

Package: `org.telegram.messenger`

## App Traits

- Cloud-based messaging (syncs across devices)
- Groups up to 200,000 members
- Channels for broadcasting
- Bots and automation
- Secret chats with end-to-end encryption
- Self-destructing messages

## Known Quirks

- **Hamburger menu**: Access to groups, channels, contacts, settings
- **Floating compose button**: Start new chat
- **Swipe right to reply**: Quick reply gesture
- **Double-tap for reaction**: Quick reaction on messages
- **Hold mic to record**: Slide up to lock recording
