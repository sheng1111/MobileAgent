# Discord Reference

Package: `com.discord`

## App Traits

- Server-based communities with text/voice channels
- Real-time messaging and voice/video calls
- Rich media: GIFs, emojis, stickers, file attachments
- Threads for organized discussions
- Screen sharing and streaming (Go Live)
- Bottom tab navigation (2025 redesign)

## Known Quirks

- **2025 redesign**: Four-tab bottom nav [Servers] [Messages] [Notifications] [You]
- **Swipe to reply**: Swipe right on message
- **Double-tap for reaction**: Quick reaction on messages
- **Long-press for options**: Reply, react, pin, delete
- **Voice channels**: Tap to join, tap again to leave
