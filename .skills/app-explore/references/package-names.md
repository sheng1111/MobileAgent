## Finding Package Names

```bash
# Search installed packages
adb shell pm list packages | grep -i <keyword>

# Get package name of foreground app
adb shell dumpsys window | grep -i mCurrentFocus

# List all packages
adb shell pm list packages
```

## Launching Apps

**MCP:**
```
mobile_launch_app with packageName: "com.google.android.youtube"
```

**Python:**
```python
from src.adb_helper import ADBHelper
adb = ADBHelper()
adb.launch_app("com.google.android.youtube")
```

## Notes

1. **Version variants**: Same app may have standard/Lite/Go versions
2. **Regional versions**: Some apps have region-specific packages (TikTok/Douyin)
3. **Rebranding**: Twitter → X but package remains `com.twitter.android`
4. **System apps**: Some Google apps are preinstalled as system apps
5. **Alternative stores**: Some apps only available in regional stores
