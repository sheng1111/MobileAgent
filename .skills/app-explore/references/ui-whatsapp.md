# WhatsApp Reference

Package: `com.whatsapp`

## App Traits

- End-to-end encrypted messaging
- Voice and video calls
- Status updates (24-hour stories)
- Group chats up to 1024 members
- Communities for organizing groups
- View Once media

## Known Quirks

- **2025 update**: Bottom tabs (moved from top)
- **Tab order**: Chats, Updates, Communities, Calls
- **Profile icon**: Now in top-right corner
- **Swipe right to reply**: Quick reply gesture
- **Hold mic to record**: Swipe up to lock
- **View Once**: Tap "1" icon when sending media
- **Message editing**: Available within time limit
