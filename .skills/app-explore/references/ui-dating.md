# Dating Apps Reference

## Common App Traits

- **Card-based discovery**: Swipe right to like, left to pass
- **Match system**: Mutual likes enable messaging
- **Profile cards**: Photos + bio + basic info (age, distance)
- **Premium features**: Super likes, rewinds, boosts, unlimited swipes

## Gesture Patterns

| Gesture | Action |
|---------|--------|
| Swipe Right | Like |
| Swipe Left | Pass/Skip |
| Swipe Up | Super Like (most apps) |
| Tap card | View full profile |
| Tap photo | Next photo |

## Known Quirks

- **Bumble**: Women message first (or "let him make first move" option since 2025)
- **Bumble**: 24-hour match expiry
- **Hinge**: Like specific photos/prompts with optional comment
- **Tinder**: Photo carousel - tap to see next
- **Soul**: Personality test matching, 3D avatars
- **Momo**: Includes live streaming features
